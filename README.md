# Bank-management-system (C++)

This is a simple Bank Management System developed using C++ and Object-Oriented Programming (OOP) concepts. The project allows users to create a bank account and perform basic banking operations such as deposit, withdrawal, and viewing account details through a menu-driven interface.

-> Features

.Create a bank account with account number and holder name
.Deposit money into the account
.Withdraw money with balance validation
.Display account details with formatted balance
.User-friendly menu-driven console interface

-> Technologies Used

.C++
.OOP Concepts (Class, Objects, Encapsulation)
.Standard Libraries: <iostream>, <iomanip>, <string>

-> Purpose

This project is designed for beginners to understand the fundamentals of C++ programming and object-oriented design by implementing a real-world banking scenario.
